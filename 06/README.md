# Session 6: EVM Foundation Recap & Roadmap

## 1. Overview
This session concludes the EVM Foundation series. The content focuses on consolidating core knowledge and preparing for the development of real-world applications (dApps).

## 2. Key Concepts to Master
Students must ensure proficiency in the following topics:

### Tools & Development Environment (Hardhat)
- **Hardhat Ignition Module**: Managing smart contract deployment.
- **Hardhat Task**: Writing automation scripts.
- **Hardhat Deploy**: Professional deployment process.

### Smart Contract Programming (Solidity)
- **ERC20**: Using `safeTransfer` to ensure secure token transactions.
- **ETH Handling**: Mechanisms for `transfer`, `receive`, `fallback`, and preventing `reentrancy` attacks.
- **Math**: Rounding principles (round up for inflow, round down for outflow).
- **Access Control**: Managing `owner`, permissions (`AccessControl`), and using `modifiers`.

## 3. Practical Exercises
- **Requirement**: Deploy the contract from Session 5 to Testnet.
- **Specific Tasks**: Write Hardhat tasks to execute the following:
  - Create tokens.
  - Buy tokens.
